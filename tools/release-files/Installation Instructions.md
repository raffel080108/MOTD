**Installation:**
1. Get UE4SS latest-experimental from https://github.com/UE4SS-RE/RE-UE4SS/releases/tag/experimental-latest (the first file under assets) and unzip it to "(Game Directory)/RogueCore/Binaries/Win64"
2. Install RogueCore Mod Manager (https://www.nexusmods.com/deeprockgalacticroguecore/mods/6)
2. Copy "MOTD" folder to "(Game Directory)/RogueCore/Binaries/Win64/ue4ss/Mods"
3. Add "MOTD: 1" to mods.txt to enable the mod
4. Copy "MOTD.pak" to "(Game Directory)/RogueCore/Content/Paks"
